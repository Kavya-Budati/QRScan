# QRScan

QRScan is a modern QR Code Generation and Scanning platform built using React, Vite, Tailwind CSS, and QR scanning libraries.

## Features

- Generate QR Codes from text or URLs
- Download QR Codes as PNG
- Download QR Codes as SVG
- Scan QR Codes using Camera
- Scan QR Codes from Uploaded Images
- Dashboard with QR History
- Protected Routes
- Login & Register UI
- Toast Notifications
- Responsive Design

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- React Router DOM

### Libraries
- qrcode.react
- html5-qrcode
- jsQR
- react-hot-toast

**QRScan**
# Overview

QRScan is a web-based platform that enables users to generate, download, and scan QR codes in real time through a responsive and intuitive interface.

# Live Demo


## Features

* Generate QR Codes from text or URLs
* Download QR Codes as PNG
* Download QR Codes as SVG
* Scan QR Codes using Camera
* Scan QR Codes from Uploaded Images
* Dashboard with QR History Tracking
* Login & Register UI
* Protected Routes
* Toast Notifications
* Responsive Design
